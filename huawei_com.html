Huawei : MateBook X Pro 2020

Huawei เปิดตัว MateBook X Pro 2020 จอสัมผัส 3K สัดส่วน 3:2 sRGB 100% ชิป Intel Gen 10 เริ่มต้นราว 51,500 บาท





ใครจะไปคิดว่าโน้ตบุ๊ค Huawei จะได้กระแสตอบรับดีขนาดสู้แบรนด์ใหญ่ๆ เจ้าตลาดได้เลย โดยในปีที่ผ่านมาก็มีการส่งออกโน้ตบุ๊ค
สูงขึ้นถึง 200% ซึ่งล่าสุด Huawei ก็ขอกระโดดมาเปิดตัวโน้ตบุ๊คสายพรีเมียมกันบ้างกับ MateBook X Pro 2020 ที่ใช้หน้าจอความละเอียด 3K,
สัดส่วน 3:2, sRGB 100% พร้อมกับ Intel Gen 10 โดยมีราคาเริ่มต้นที่ 1,499 ยูโรหรือประมาณ 51,500 บาท





สเปคเบื้องต้น MATEBOOK X PRO 2020

1. CPU : Intel Core i5-10210U / i7-10510U
2. การ์ดจอ : Intel UHD / NVIDIA GeForce MX250
3. RAM : 8GB/ 16GB LPDDR3 2133 MHz
4. Storage : SSD m.2 PCIe 512GB/ 1TB
5. หน้าจอ : 13.9″ 3000×2000, 3:2, 450 nits, 10-point multi-touch
6. Network : WiFi 802.11ac / Bluetooth 5.0 / NFC
7. Interface : USB-C x2 (data, PD+DP), USB 3.0-A x1, 3.5 mm Headset
8. Battery : 56 Wh up to 15 Hours
9. Camera : 1MP
10. ระบบเสียง : Speaker x4, Microphone x4
11. ขนาด : 304 x 217 x 14.6 mm
12. น้ำหนัก : 1.33 kg
13. OS : Windows 10 Home + Microsoft Office 365

วัสดุตัวเครื่อง MateBook X Pro 2020 ทำมาจาก Aluminum alloy ทั้งตัวเครื่อง สวยหรูระดับพรีเมียม
แบบ Uni-Body ขอบตัด Diamond Cut สุดประณีต พร้อมใช้หน้าจอขอบบาง Fullview Display สัดส่วนหน้าจอต่อบอดี้
ตัวเครื่องสูงถึง 91% ซึ่งถือเป็นรุ่นแรกของโลกที่ทำได้เลยทีเดียวที่มีพื่นที่หน้าจอเยอะขนาดนี้ โดยเปิดตัวมาจะมีจำหน่ายด้วยกัน 3 สี
คือ เขียว Emerald Green, เทา Space Gray และ เงิน Mystic Silver



ฟีเจอร์เด่นที่โน้ตบุ๊ค Huawei เกือบทุกรุ่นต้องมีคือ Huawei Share คือใช้มือถือวางบนจุดเชื่อมต่อ NFC บนคีย์บอร์ด
เพื่อเชื่อมต่อใช้งานแบบไร้สายได้ ซึ่งสามารถทำได้ทั้งการแชร์หน้าจอ โอนไฟล์ ย้ายไฟล์ หรือใช้คีย์บอร์ดของโน้ตบุ๊คเพื่อพิมพ์ข้อความ
บนมือถือก็ได้เช่นกันสะดวกสบายสุดๆ แต่ใช้ได้เฉพาะมือถือของ Huawei เท่านั้น





สำหรับปุ่มเปิดปิดหน้าจอจะยังคงดีไซน์แบบเดิมคืออยู่ที่มุมบนขวา พร้อมกับมีสแกนลายนิ้วมือในตัวด้วย ส่วนกล้อง Webcam
ก็ซ่อนไว้อยู่ใต้คีย์บอร์ดเหมือนเดิมอยู่ระหว่างปุ่ม F6 และ F7
ลำโพงตัวเครื่องให้มา 4 ตัวเสียงดังจัดเต็ม มีทั้งวูฟเฟอร์ที่เป็นเสียงเบสให้ความลึกและหนักแน่น ผสานกับทวีตเตอร์ให้เสียงแหลมใส
กังวานชัดเจน รวมถึงไมโครโฟนก็ให้มา 4 ตัวแบบตัดเสียง สามารถรับเสียงของพูดได้ไกลสูงสุดถึง 4 เมตรด้วยกัน เวลาวิดีโอคอลจึงไม่จำเป็นต้องใส่
Smalltalk หรือนั่งติดตัวเครื่องจนเกินไป





ทางด้านพอร์ตเชื่อมต่อตัวเครื่องฝั่งทางซ้ายจะเป็น USB Type C 2 ช่อง โดยจะมีช่องที่เป็น DP+PD 1 ช่องและธรรมดา 1 ช่องด้วยกัน
พร้อมมีรูเสียบ Headset 3.5 มาให้ปกติ ส่วนฝังทางขวาจะเป็น USB Type A ปกติ 1 ช่อง แต่แอบเสียดายที่ตัวเครื่องไม่ได้มี Thunderbolt 3 มาให้
ส่วนอะแดปเตอร์ชาร์จก็ให้มาเป็นแบบ PD 65W ขนาดพอๆ กับที่ชาร์จมือถือ พกพาไปไหนมาไหนค่อนข้างสะดวก





สำหรับระบบตัวเครื่องจะมาพร้อมกับ Windows 10 Home แท้ในตัว พร้อมกับ Microsoft Office 365 แบบ Pre-installed มาให้ด้วย

ซึ่งบ้านเราก็มีตัวMateBook D15 ขายอยู่ที่ราคาเริ่มต้น 17,900 บาท ถูกกว่าที่เปิดตัวภายในงานที่เริ่มต้น 649 ยูโรหรือประมาณ 22,500 บาท
